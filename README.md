# Personal Introduction Python Program 

This is a simple Python program that asks the user for their name, age, and hobby, then displays a friendly welcome message.

## Features
- Uses `input()` to collect user information
- Stores data in variables
- Displays a welcoming message using `print()`
- Beginner-friendly Python project

## Files Included
- `personal_intro.py` – Main Python program
- `requirements.txt` – Project dependencies
- `screenshot.png` – Output screenshot

## How to Run

1. Make sure Python is installed.
2. Run the program:

```bash
python personal_intro.py
